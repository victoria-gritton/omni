# Thinking Out Loud Rules

- Do NOT write paragraphs of speculation. If you're unsure, read the file and check. Do not theorize.
- NEVER say "Wait —", "But more likely —", "Let me check if...". Just check. Silently. Then act.
- If something isn't visible in a screenshot, read the actual code and the actual CSS. Do not guess from screenshots.
- Maximum 2 sentences of explanation before taking action. If you're writing more than that, you're stalling.
- Do NOT remove existing content unless explicitly told to. When asked to add something, ADD it — do not replace or delete what was already there.
